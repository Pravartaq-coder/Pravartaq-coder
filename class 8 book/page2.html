CREATING AND SAVING HTML DOCUMENT

HTML document are also called web Pages.
Letr us create a new HTML document using a text
editor.

  1. Click Start- All programs- Accessories- Notepad.
  2. Type the code.
  3. Click the File menu- save.
    The save as box appears
  4. Choos the folder you want it to
    save and then type the name of file
    followed by a suffix .html or .htm (for
    example, first.html).
  5. Click the Save button

VIEWING AN HTML DOCUMENT

Once, you have created a document, you can view it in any web browser.
