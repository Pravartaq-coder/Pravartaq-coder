What is HTML?

HTML (HyperText Markup Language) is a language used to create web page. HTML tags to create the structure of a web page. These tags tell the browser how to display text or graphics.

Features OF HTML

  • HTML documents are simple text files.
  • There is no need of compiling the program as in the case of other programming languages.
  • It is not a case sensitive language, hence, there is less chance of errors.

HTML EDITOR

There are two kind of HTML EDITORS.-

  1. WYSIWYG editors:These are GUI based programs that let you create web page without knowing the HTML code. Websites can be created quickly using these editors WYSIWYG stands for What You See is what You Get. Examples of these editors MS Frontpage and Adobe Dreamweaver.
  2. Text edifor: These are programs that let you create web page by writing the HTML code. You need to know the correct usage of the tags. Example of these editors Notepad and WordPad.
